Dieses Projekt merged mit Hilfe des PDF24 Tools eine Zugferd XML Datei mit einer PDF Datei zu einer PDF/A Zugferd Datei im xrechnung:cii Format.

## Installation

Das Tool setzt eine Installation von PDF24 voraus. Das Tool kann über den folgenden Link heruntergeladen werden:

https://tools.pdf24.org/de/creator#download

## Voraussetzungen

Das Tool PDF24 muss im Pfad von Windows, bzw. dem aktuellen Benutzer liegen.

## Nutzung

- Starten
- PDF Datei auswählen
- Zugferd XML Datei auswählen
- Zieldatei auswählen (Standartmäßig wird hier der Dateiname der PDF Datei mit dem Suffix _zugferd_YYYYMMDD_HH24MISS.pdf vorausgewählt)
- Dateien verarbeiten klicken

Heraus kommt eine PDF/A Datei im xrechnung:cii Format.

Diese kann validiert werden mit einem Zugferd Validator wie zum Beispiel:

https://www.portinvoice.com/
